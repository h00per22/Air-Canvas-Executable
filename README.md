# Air Canvas (Executable Version)

This repository contains the executable (.exe) version of the **Air Canvas** project.

⚠️ **Note**: The source code is not included in this repository.  
This project is shared only for **demonstration and execution purposes**.

## About the Project
Air Canvas allows users to draw in the air using hand gestures captured via a camera.

## How to Run
1. Download `Air Canvas.exe`
2. Double click the file
3. Allow camera access if prompted

## System Requirements
- Windows OS
- Webcam
- Python runtime bundled in exe
- Antivirus may warn (false positive)

## Technologies Used
- Computer Vision
- Hand Gesture Recognition
- OpenCV / MediaPipe (compiled)

This executable is provided for educational/demo purposes only.
